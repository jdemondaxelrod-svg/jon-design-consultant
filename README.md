# Jon DeMond-Axelrod — Portfolio

Single-page portfolio for residential design-build and consultative sales consulting.

**Production URL:** [www.jondemondaxelrodstudio.com](https://www.jondemondaxelrodstudio.com)

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Production build

```bash
npm run build
npm run start
```
